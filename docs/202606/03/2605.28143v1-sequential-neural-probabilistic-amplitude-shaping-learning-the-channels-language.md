## 摘要
我们提出了第一个神经概率幅度整形方法，在考虑所有实现损耗的同时优于现有方法，该方法使用无块、易于实现的序列自回归编码器，与算术分布匹配兼容，从而降低了速率损耗并实现了更高的可达信息速率。

## Abstract
We present the first neural probabilistic amplitude shaping that outperforms existing methods while accounting for all implementation losses, using a block-less, easily implementable sequential autoregressive encoder compatible with arithmetic distribution matching, yielding reduced rate loss and higher achievable information rates.

## 1. 核心问题与整体含义（研究动机和背景）
**研究动机**：概率幅度整形（PAS）在非线性光纤信道中可同时获得线性和非线性整形增益，但现有方法存在根本性矛盾：
- 短块长（如枚举球面整形ESS）能降低非线性噪声，但速率损失（rate loss）增大。
- 序列选择方法通过多次生成候选序列并选取低失真者，但需要多次评估、额外信令开销，且计算复杂度高。
- 已有的神经整形方法（NPS/NPAS）采用块状结构，只能捕捉块内依赖，忽略跨块相关性，且训练目标未显式考虑由符号依赖引入的速率损失，导致学习到高表达力的时间结构却带来过大的速率损失，抵消甚至逆转可达信息率（AIR）增益。

**本文意义**：首次提出一种在实际实现中考虑所有损失（速率损失、信道匹配损失）的神经PAS方案，通过无块化序列自回归编码器与速率损失感知的训练目标，实现了优于现有方法的性能，为非线性信道下的概率整形提供了新范式。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
将未编码符号序列建模为**有限记忆的平稳自回归过程**，类似语言模型“预测下一个token”，并显式将速率损失纳入训练目标，平衡非线性容忍度与整形效率。

### 关键技术细节
1. **序列自回归模型（Seq-NPAS）**：
   - 联合分布分解：\( p_\theta(\mathbf{a}) = p_\theta(a_1^{\mu-1}) \prod_{t=\mu}^N p_\theta(a_t | a_{t-1}^{t-\mu+1}) \)，其中 \(\mu\) 为记忆长度。
   - 神经网络架构：单层 Transformer（8个注意力头，隐藏维64，前馈维256，旋转位置编码，Swish-gated线性单元）。
2. **可微训练**：
   - 使用Gumbel-Softmax重参数化+直通估计器实现离散采样的梯度传播。
   - 信道模拟：可微的单通道单偏振扰动信道模型。
3. **速率损失感知的训练目标（\(L^{++}\)）**：
   \[
   L^{++} = L + R_{\text{loss}} + \lambda D_{\text{KL}}(p(x) \| p_{\text{MB}}(x))
   \]
   - \(L\)：原有调整二进制交叉熵损失（含信道不匹配KL散度）。
   - \(R_{\text{loss}}\)：理论速率损失，即边际熵之和与熵率之差（公式2），可微。
   - 正则项：鼓励边际分布接近优化后的Maxwell-Boltzmann分布（\(\lambda\) 控制权衡）。
4. **推理**：与算术分布匹配（ADM）兼容，生成符号序列。

### 算法流程（文字说明）
- 训练阶段：神经网络接收前 \(\mu-1\) 个符号，输出当前符号的类别分布；通过Gumbel-Softmax采样得到符号；送入可微信道模型，产生LLR；计算损失 \(L^{++}\)，反向传播更新参数。
- 推理阶段：将训练好的网络与ADM结合；ADM根据网络预测的分布将信息比特映射为符号序列；传输后经实际接收机处理，计算AIR。

## 6. 主要结论与发现
1. **速率损失感知训练是关键的**：忽略速率损失（如原始NPAS）会导致高内在速率损失，抵消非线性增益；加入 \(R_{\text{loss}}\) 项后显著降低速率损失。
2. **Seq-NPAS++ 提供了最佳AIR**：在非线性高功率区域，比ESS和序列选择高约0.05 bits/2D符号；在低功率区域略逊于ESS（因额外时间结构引入速率损失而非线性较弱）。
3. **无块化设计优势**：Seq-NPAS的平稳自回归模型避免了块边界伪迹，兼容ADM，易于实现。
4. **首次实现考虑所有实现损失的神经PAS胜过现有方法**：为非线性信道概率整形提供了实用且高性能的方案。

## 7. 优点
- **方法创新**：将语言模型中的自回归范式引入概率整形，建模长程依赖且无块限制。
- **训练目标显式处理速率损失**：解决了先前神经整形中忽略实现损失的缺陷，公式可微，便于优化。
- **实用性强**：与ADM天然兼容，无需额外信令或多次评估，实现简单。
- **性能优越**：在计入所有实际损耗后仍超越现有最佳方法（包括序列选择），验证了其实际价值。
- **可推广性**：方法不限于光纤Kerr非线性，可适用于其他有记忆非线性信道。
