## 摘要
错误纠正码能够实现可靠通信，但在不同编码族和码长下，实用的软解码仍具挑战性。我们提出SB-ECC，一种基于分数的解码器，将解码转化为连续时间去噪过程。神经去噪器定义了一个概率流常微分方程（ODE），在奇偶约束的引导下，迭代地将含噪声的信道观测更新为有效码字。该模型无需时间/SNR条件即可在各种噪声水平下训练，从而无需SNR估计即可进行推理，并通过ODE求解器预算控制直接的延迟-准确度权衡。我们使用原始的带符号信道观测作为输入，以学习连续的去噪场。在42种码/SNR设置中，SB-ECC在39/42个条目上实现了最佳误码率（BER），与最强竞争基线相比，平均SNR增益为0.17dB，最大增益为0.46dB。我们展示了将求解器从Euler切换为DPM可在保持-ln(BER)的同时，将端到端解码时间平均减少8.86%（最高达12.82%）。

## Abstract
Error-correcting codes enable reliable communication, yet practical soft decoding remains challenging across code families and block lengths. We propose SB-ECC, a score-based decoder that casts decoding as continuous-time denoising. A neural denoiser defines a probability-flow ordinary differential equation (ODE) that iteratively updates the noisy channel observation toward a valid codeword, guided by parity constraints. The model is trained across noise levels without time/SNR conditioning, enabling inference without SNR estimation and supporting a direct latency accuracy trade off controlled by the ODE solver budget. We use the raw signed channel observation as input for learning a continuous denoising field. Across 42 code/SNR settings, SB-ECC achieves the best BER in 39/42 entries, with an average SNR gain of 0.17dB and a maximum gain of 0.46dB over the strongest competing baseline, we showed that swapping the solver from Euler to DPM preserves -ln(BER) while reducing end-to-end decoding time by 8.86% on average (up to 12.82%).

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：错误纠正码（ECC）的软解码在实际中面临挑战，尤其在不同码族和码长下，最大似然（ML）解码是NP-hard的。传统迭代解码器（如置信传播BP）近似性好但表达受限；基于深度学习的方法分为模型驱动（如unrolled BP）和模型无关（如Transformer解码器），但存在样本复杂度过高、需依赖SNR估计等问题。
- **动机**：现有模型无关解码器通常预处理为仅保留幅度（|y|）并丢弃方向信息，这限制了连续去噪场的几何学习。同时，许多方法需要SNR作为输入或依赖特定时间步，缺乏灵活性。
- **整体含义**：提出一种基于分数（score-based）的解码器SB-ECC，将解码视为连续时间去噪，直接利用带符号的信道观测，无需SNR估计，并支持通过ODE求解器步数灵活调节延迟-精度权衡。

## 2. 方法论
### 核心思想
- 将信道接收信号 `y = xs + z`（其中`xs`为BPSK调制码字，`z`为AWGN噪声）视为一种方差爆炸（VE）扩散过程在未知噪声水平的观测。
- 训练一个神经去噪器 `ˆϵθ(y, s(y))` 预测添加的噪声，从而近似分数 `∇x log pt(x)`。
- 解码时，通过求解概率流常微分方程（PF-ODE）从`y`反向积分，得到干净码字估计；积分过程由奇偶约束（通过硬判决的校正子syndrome引导，并可提前停止。

### 关键技术细节
- **输入**：直接使用带符号的`y`（而非|y|），保留方向信息。
- **架构**：基于CrossMPT（交叉注意力消息传递Transformer），将变量节点（VN）和校验节点（CN）作为两种模态，通过Tanner图掩码注意力进行信息交换。输出为连续去噪方向`ˆϵθ(y, s(y)) ∈ Rⁿ`。
- **训练目标**：标准噪声预测损失 `Lϵ = E[∥ˆϵθ(y, s(y)) - ϵ∥²]`，其中`y = x₀ + σ(t)ϵ`，`t`均匀采样于[0,1]，`σ(t) = σ_min + (σ_max - σ_min)·t`为线性噪声计划。
- **时间无条件**：模型不依赖时间或SNR输入，消除了推理时SNR估计需求。
- **推理过程**：
  - 从`x(0)=y`开始，在σ空间均匀离散，使用Euler或DPM-Solver更新：`x(i+1) = x(i) - Δσ · ˆϵθ(x(i), s(x(i)))`。
  - 每一步后计算硬判决的校正子，若为零则提前停止输出码字。
  - 步数`Nsteps`可在测试时调整，直接控制延迟-精度。

## 7. 优点
- **创新性**：将score-based扩散模型应用于解码，直接使用带符号输入，保留连续去噪几何，无需SNR估计。

- **性能卓越**：在多种码族和SNR下大幅超越现有模型无关解码器（包括DDECC和CrossMPT），接近或超过BP千次迭代表现。

- **实用性**：支持提前停止（早期退出），实际平均迭代次数少，推理开销有限。
