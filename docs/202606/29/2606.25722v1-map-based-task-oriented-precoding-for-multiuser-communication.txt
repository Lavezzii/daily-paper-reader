# MAP-Based Task-Oriented Precoding for Multiuser Communication 

Mohammad Javad Ahmadi, Rafael F. Schaefer Senior Member, IEEE ,and H. Vincent Poor, Life Fellow, IEEE 

Abstract —We propose a task-oriented multiuser wireless com-munication framework for distributed classification based on a MAP-driven system design under wireless channel impairments. By deriving a tractable class-mean separation objective, the pro-posed approach enables low-complexity design of both learning-based feature extraction and precoding strategies. Unlike exist-ing covariance-based and reconstruction-oriented methods, the proposed formulation avoids repeated covariance inversions and eigen-decomposition operations while directly improving class separability after channel distortion. Simulation results demon-strate that the proposed method achieves higher classification accuracy than existing schemes, while simultaneously reducing computational complexity. 

Index Terms —Task-oriented communication, MAP detection, feature learning, wireless precoding. 

I. I NTRODUCTION 

Task-oriented and semantic communication has recently emerged as a promising paradigm for next-generation wireless systems, where the goal is to reliably convey task-relevant se-mantic information rather than reconstructing raw transmitted signals [1]. In contrast to conventional communication systems that primarily focus on minimizing symbol or signal distortion, semantic and task-oriented frameworks aim to optimize end-task performance such as classification or inference accuracy at the receiver. This shift has motivated the integration of machine learning techniques with wireless communication design, enabling joint optimization of feature representation, transmission strategies, and decision-making under channel impairments. As a result, new system designs are required that account for both learning-based feature extraction and channel-aware signal processing in a unified framework [2]–[8]. These methods mainly optimize task perfor-mance indirectly through representation learning frameworks such as deep joint source–channel coding (JSCC) or informa-tion bottleneck models. More recent approaches further shape the learned feature space to enhance post-channel inference performance, rather than solely compressing representations for transmission [5]–[8]. Although achieving promising per-formance, these methods still rely on conventional commu-nication objectives that are not explicitly aligned with end-task inference. For instance, [5] targets accurate signal recon-struction via the linear minimum mean-squared error criterion, while [6], [7] employ mutual information as the optimization objective, focusing on semantic-aware representation learning rather than task-specific discrimination. The method in [8] employs maximal coding rate reduction (MCR 2) to enhance class separability in the learned feature space. This reliance on criteria that are not explicitly aligned with task-specific inference objectives limits the effectiveness of existing task-oriented communication systems. To address this issue, this paper proposes a task-oriented wireless communication framework based on an approximate MAP error formulation in the channel-affected feature space, which is more aligned with the end-task objective than conven-tional approaches in the literature. Unlike existing approaches relying on reconstruction-based or surrogate coding-rate ob-jectives, the proposed formulation directly targets post-channel MAP decision performance while avoiding expensive covari-ance and matrix decomposition operations. Simulation results demonstrate improved classification accuracy and more stable performance compared to existing methods, while significantly reducing computational complexity. Feature Extractor Complexity : The proposed objective in (12) requires computing pairwise distances between class-mean feature vectors across C classes in a D-dimensional space, which results in a computational complexity of O(DC 2).In contrast, the MCR 2-based objective in [8] requires the computation of empirical covariance matrices over a minibatch of size B, together with the evaluation of log-determinant terms over C + 1 covariance matrices of size D × D. This leads to a computational complexity of O(BD 2 + CD 3),which is significantly higher than the proposed approach in high-dimensional settings. We note that, to analyze the feature extractor complexity, we ignore the computational cost of the neural network layers, as they exhibit standard complexity in both the proposed method and MCR 2 and are not the focus of this work. IV. N UMERICAL RESULTS 

The performance of the proposed task-oriented commu-nication framework is evaluated using our implementation available on GitHub [9]. Unless stated otherwise, the following settings are used throughout the simulations. We consider a distributed wireless classification system using the CIFAR-10 dataset with K = 2 views per sample, where each view is generated using stochastic data augmentation including ran-dom resized cropping, horizontal flipping, and color jittering, resulting in two independently augmented observations per input image. Each view is encoded using a dedicated ResNet-18 backbone initialized with ImageNet pretrained weights, followed by a two-layer fully connected projection head that maps the 512-dimensional feature representation to a 2Dk-dimensional latent embedding. For performing the update equation in (14), we select η = 0 .1, τ = 0 .7, and I0 = 30 .The wireless channel coefficients are generated independently according to CN (0 , 1) , corresponding to a Rayleigh fading environment, with transmit power P = 2, noise variance 

σ2 = 1 , and system parameters Nk = 6 and T = 1 .In Fig. 1, we plot the test accuracy versus the number of training epochs for the proposed feature extractor and MCR 2

in [8] with different values of ϵ. For a fair comparison, both methods use the same network architecture. It can be observed that the proposed method closely matches the performance and convergence behavior of MCR 2 across training epochs, while requiring significantly lower computational complexity as discussed in Sec. III-B3, due to its independence from covariance estimation and matrix decomposition operations. Moreover, the results indicate that MCR 2 exhibits strong sensitivity to the choice of ϵ, where the best performance is achieved at ϵ = 5 , while both smaller and larger values lead to noticeable performance degradation. In contrast, the proposed method does not rely on such sensitive tuning parameters, which improves its robustness. We compare the performance of different precoding schemes in Fig. 2, which illustrates the MAP detection error performance with respect to the number of receive antennas. The baselines are the MCR 2 precoder proposed in [8] and the LMMSE baseline in [5] (with modifications detailed in Appendix A of [8]). For a fair comparison focused solely on precoder design, all considered schemes employ the same pretrained MCR 2-based feature extractor [8]. It is observed that the proposed precoder consistently achieves lower MAP error compared to both the LMMSE and MCR 2 precoders. This superior performance stems from the fact that the pro-posed method directly minimizes an approximation of the MAP detection error. In contrast, the LMMSE precoder is designed to minimize the mean-squared error of feature recon-struction at the receiver, while the MCR 2 precoder focuses on maximizing rate-related objectives that are not directly aligned with classification error minimization. In addition, it is observed in Fig. 2 that the computational complexity of the MCR 2 precoder is significantly higher than that of both the proposed and LMMSE precoders. C ONCLUSION 

In this paper, we have proposed a MAP-driven task-oriented communication framework for distributed wireless classifi-cation. A low-complexity feature learning objective and a gradient-based precoder were derived from an approximation of the MAP detection error. Numerical results show that the proposed method achieves higher classification accuracy, lower computational complexity, and improved robustness compared to conventional precoding schemes.
