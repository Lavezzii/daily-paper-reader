## 摘要
准确的信道状态信息（CSI）预测对于5G大规模MIMO系统中的主动波束赋形和资源管理至关重要，然而在基站硬件上部署基于Transformer的高精度预测器仍然具有挑战性，因为最强大的模型拥有超过3000万个参数。本文介绍了一种轻量级PCGAE-Net，它不通过事后压缩来解决效率问题，而是纠正了当前技术中的两个架构缺陷。第一个是顺序注意力排序偏差：在CS3T-UNet中，分组时间注意力（GTA）始终作用于已经由十字形空间注意力（CSA）变换过的特征，扭曲了GTA能够捕获的时间信息。我们通过将两个注意力模块路由到相同的层归一化输入，并通过学习的逐通道sigmoid交叉门（CrossGate）组合它们的独立输出来消除这种依赖性。第二个缺陷是未压缩的瓶颈：在最深的编码器阶段应用全自注意力，其中通道深度达到4C，代价呈二次增长且携带冗余特征。使用1×1卷积的瓶颈自编码器（BAE）将深度减半，并使用辅助重建损失来防止信息崩溃。将这些组件封装在更浅的编码器-解码器中，并采用频域降维（N_f=32，C=48），得到的模型仅具有854万个参数——比CS3T-UNet基线减少58%——在QuaDriGa数据集上的单步预测中，在5 km/h时性能提高了3.26 dB，在9 km/h时提高了6.0 dB。

## Abstract
Accurate channel state information (CSI) prediction is essential for proactive beamforming and resource management in 5G massive MIMO systems, yet the deployment of high-accuracy transformer-based predictors on base-station hardware remains challenging because the most capable models carry upwards of 30\,M parameters. This paper introduces Lightweight PCGAE-Net, which addresses the efficiency problem not by post-hoc compression but by correcting two architectural flaws in the current state of the art. The first is a sequential attention ordering bias: in CS3T-UNet, group-wise temporal attention (GTA) always operates on features that have already been transformed by cross-shaped spatial attention (CSA), distorting what temporal information GTA can capture. We remove this dependency by routing both attention modules to the same layer-normalized input and combining their independent outputs through a learned per-channel sigmoid CrossGate. The second flaw is an uncompressed bottleneck: applying full self-attention at the deepest encoder stage, where channel depth reaches $4C$, is quadratically expensive and carries redundant features. A Bottleneck AutoEncoder (BAE) with $1\times1$ convolutions halves this depth and uses an auxiliary reconstruction loss to prevent information collapse. Wrapping these components inside a shallower encoder-decoder with frequency-domain dimensionality reduction ($N_f\!=\!32$, $C\!=\!48$) produces a model with just 8.54\,M parameters -- 58\% fewer than the CS3T-UNet baseline -- that outperforms it by up to 3.26\,dB at 5\,km/h and 6.0\,dB at 9\,km/h in single-step prediction on QuaDriGa dataset.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：5G 大规模 MIMO 系统中，基站需要精确的 CSI 预测以提前计算波束赋形权值、容忍反馈延迟。近年来 Transformer 模型（如 CS3T‑UNet）在信道预测任务上达到顶尖性能，但其参数规模庞大（超过 3000 万），部署在基站硬件上具有挑战性。
- **问题**：现存高精度模型并非因容量不足而是因架构缺陷导致效率低下，即**顺序注意力偏差**（CSA 先于 GTA 执行，限制了 GTA 的时间信息提取）和**未压缩的瓶颈**（最深层通道深度达 4C，全自注意力计算昂贵且特征冗余）。事后压缩（剪枝、量化）无法从根本上解决这些架构问题。
- **研究目标**：设计一种轻量级网络，在不显著牺牲准确率的前提下大幅减少参数量，同时通过架构修正提升预测性能，从而适用于实际 5G 基站部署。

## 2. 论文提出的方法论

- **核心思想**：从架构层面修正 CS3T‑UNet 的两大缺陷，采用并行 CrossGate 注意力（PCG）和瓶颈自编码器（BAE），并结合浅层编码‑解码与频域降维。
- **关键技术细节**：
  - **并行 CrossGate 注意力块（PCG Block）**：
    - CSA 和 GTA 接收同一层归一化（LN）后的输入，独立计算特征图 \(S\) 和 \(T\)。
    - 通过可学习的逐通道 Sigmoid 门控（CrossGate）融合二者：  
      \( g = \sigma(W_g [S; T] + b_g) \)，\( F = g \odot S + (1-g) \odot T \)。
    - 初始化 \(W_g=0, b_g=0\)，使得初始 \(g=0.5\)，保证训练平稳。
    - 最终输出经残差连接和 FFN：\( Y = X + F + \text{FFN}(\text{LN}(X+F)) \)。
    - 消除了顺序偏差，让网络自适应选择时空信息比例。
  - **瓶颈自编码器（BAE）**：
    - 在最深编码器阶段（8C 通道），先用 1×1 卷积压缩至 4C 通道（\(Z = \text{GELU}(\text{GN}(\text{Conv}_{1\times1}(X)))\)），再通过 1×1 卷积重建回 8C（\(\hat{X} = \text{GN}(\text{Conv}_{1\times1}(Z))\)），并添加残差连接。
    - 压缩后的特征 \(Z\) 传递给一个更小的注意力模块（4C 深度），自注意力成本降低约 75%。
    - 引入辅助重建损失 \(L_{\text{rec}} = \|\hat{X} - X\|_F^2\) 与预测损失联合训练：\(L = L_{\text{pred}} + \alpha L_{\text{rec}}\)，\(\alpha = 0.1\)，防止瓶颈压缩丢失预测所需的结构信息。
  - **浅层编码‑解码与降维**：
    - 将编码‑解码的 Merge/Expand 块对数从 3 对减至 2 对（去除最深层）。
    - 延迟域裁剪：保留 ADP 域中最强能量最高的 32 个延迟 bin（\(N_f=32\)），输入尺寸减半。
    - 基础通道宽度从 \(C=64\) 降至 \(C=48\)。
    - 最终模型仅 854 万参数。
