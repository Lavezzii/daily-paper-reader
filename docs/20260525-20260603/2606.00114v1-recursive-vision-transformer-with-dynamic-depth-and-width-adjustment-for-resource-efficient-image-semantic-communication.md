## 摘要
图像语义通信是下一代无线通信系统的关键组成部分。然而，此类系统通常占用大量内存且计算复杂度高，难以部署在资源受限的设备上。为解决这些挑战，我们提出了一种基于视觉Transformer（ViT）的图像语义通信系统。在该系统中，引入递归结构以迭代优化语义特征并减少参数数量。此外，设计了三种动态调整策略以自适应降低计算复杂度：动态深度调整、动态宽度调整以及宽度-深度联合优化。动态深度调整根据图像内容和信道条件自适应确定递归模块的数量，而动态宽度调整则选择性保留重要神经元和注意力头。宽度-深度联合优化进一步实现了灵活的计算配置。仿真结果验证，所提出的基于递归ViT的系统结合三种动态调整策略，在相当的计算复杂度下，参数数量减少了48.7%，并实现了比现有基线更高的重建质量。

## Abstract
Image semantic communication is a critical component in next-generation wireless communication systems. However, such systems typically suffer from large memory footprints and high computational complexity, making them difficult to deploy on resource-constrained devices. To address these challenges, we propose a vision transformer (ViT)-enabled image semantic communication system. In this system, a recursive structure is introduced to iteratively refine semantic features and reduce the parameter count. In addition, three dynamic adjustment strategies are designed to adaptively reduce computational complexity: dynamic depth adjustment, dynamic width adjustment, and joint width-depth optimization. Dynamic depth adjustment adaptively determines the number of recursive modules according to image content and channel conditions, while dynamic width adjustment selectively preserves important neurons and attention heads. The joint width-depth optimization further enables flexible computation configurations. Simulation results verify that the proposed recursive ViT-based system, combined with the three dynamic adjustment strategies, reduces the parameter count by 48.7% and achieves higher reconstruction quality than existing baselines under comparable computational complexity.

## 2. 方法论

### 2.1 核心思想
- 采用**递归Transformer单元（RTU）** 替代传统堆叠的多个Transformer层：每个RTU内部共享同一组参数，并递归应用两次，实现与L层相同有效深度但参数减少50%。
- 提出三种动态调整策略：
  1. **动态深度调整（DD）**：基于累计停止分数（cumulative stopping score）逐patch决定是否提前终止计算，实现内容感知的深度控制。
  2. **动态宽度调整（DW）**：基于可学习重要性分数和信道自适应阈值，对神经元和注意力头进行结构化剪枝，实现信道感知的宽度控制。
  3. **联合宽度-深度优化（Joint）**：先宽度后深度，在稳定宽度结构上训练深度控制器，避免耦合干扰。

### 2.2 关键技术细节
- **递归Transformer单元（RTU）**：
  - 每个RTU包含一个Transformer编码层，该层被递归调用两次。
  - 第一次递归提取高级语义特征，第二次递归强化特征一致性并整合。
  - 通过L/2个RTU实现L层效果，参数量减少约48.7%。
- **动态深度调整**：
  - 为每个patch维护累计分数`$\tilde{h}_m^{(l)}$`，基于特征变换幅度和信道噪声计算停止分数`$h_m^{(l)}$`。
  - 当累计分数超过阈值T时，该patch停止计算，被前向跳过。
  - 综合所有层的加权输出作为最终表示，使用可微的ponder损失`$L_{\text{ponder}}$`衡量平均计算深度。
  - 总损失`$L = L_{\text{MSE}} + \tau L_{\text{ponder}}$`。
- **动态宽度调整**：
  - 对每个可剪枝单元（神经元或注意力头），学习重要性向量`$a$`和阈值`$\xi$`。
  - 保留比例`$K(\xi)=100 \cdot \text{sigmoid}(\gamma\xi + \delta\sigma^2)$`，信道越差保留越多。
  - 通过Straight-Through Estimator（STE）反向传播，稀疏正则化`$L_\rho$`确保达到目标剪枝率`$\rho_t$`。
- **联合优化**：先训练宽度剪枝得到稳定结构，再固定宽度参数，训练深度控制器（此时去除停止分数中的信道项）。

### 2.3 算法流程
- **动态深度**（Algorithm 1）：初始化所有patch为active，逐层处理，对active patch用当前RTU更新特征并计算停止分数，累计分数超过T则标记为stopped，最终输出所有层的加权和。
- **联合优化**（Algorithm 2）：对候选目标剪枝率`$\rho_t$`进行宽度剪枝，得到剪枝后的骨干网络；再对每个候选停止阈值T，冻结宽度参数，微调深度控制器；最后评估不同`$(\rho_t, T)$`组合的性能-复杂度权衡。
